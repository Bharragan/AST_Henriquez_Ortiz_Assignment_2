# Homework 2 – Time Series Analysis

This repository contains the implementation and results for Homework 2 of the Time Series Analysis course (Second Semester 2025).

---

## Repository Structure

```
project_root/
├── data/            # Raw and processed datasets
├── notebooks/       # Jupyter notebooks with code and analysis
├── results/         # Figures and tables generated from experiments
├── src/             # Utility functions and plotting scripts
└── docs/            # PDF
```

---

## Main Tasks

* Simulation of ARMA processes
* Analysis of stationarity
* ACF and PACF (theoretical and sample)
* Model estimation and forecasting
* Evaluation of uncertainty and residual diagnostics

---

## Dependencies

Install required packages:

```
pip install -r requirements.txt
```

---

## Reproducibility

All notebooks are fully reproducible.
Parameters are fixed using a global random seed.
